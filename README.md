# MUFFIGOUT APPAREL HUB

Official storefront for MUFFIGOUT APPAREL HUB — men's clothing, crafted with trust, worn with pride.

🌐 **Live site:** https://muffigoutapparelhub.com

## Tech Stack

- React + Vite + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (database, auth, edge functions, storage)
- Capacitor (iOS / Android shells)
- PayU (payments) + Delhivery (logistics)

## Local Development

Requires Node.js 18+ and npm.

```sh
npm install
npm run dev
```

The app runs on `http://localhost:8080` by default.

## Build

```sh
npm run build
npm run preview
```

## Project Structure

- `src/pages` — route-level pages
- `src/components` — shared UI and feature components
- `src/hooks` — reusable React hooks
- `src/integrations/supabase` — auto-generated Supabase client + types
- `supabase/functions` — backend edge functions
- `supabase/migrations` — database schema migrations
